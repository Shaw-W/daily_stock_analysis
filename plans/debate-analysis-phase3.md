# 第三阶段计划：高级辩论能力与可配置策略

## 目标

在独立辩论链路和 Web 展示成熟后，引入更高级的可配置 Agent、质量评估和批量聚合能力。

## 重点任务

1. Agent 策略可配置：允许启停单个 Agent、调整权重、选择 prompt 模板版本。
2. 辩论质量评估：记录数据覆盖率、Agent 失败率、裁判一致性和历史回测表现。
3. 批量聚合报告：对一组股票生成辩论榜单、分歧榜和高置信机会/风险列表。
4. 可选混合分析模式：提供显式开关，在普通分析后追加辩论摘要或生成对照报告。
5. 成本控制：增加预算、token 上限、分阶段跳过和缓存复用策略。
6. 评估闭环：将辩论结论纳入回测与校准，沉淀更可靠的 Agent 权重。

## 不做

- 不在未明确开启时增加普通分析默认成本。
- 不引入未经验证的数据源替代现有 fallback 体系。
